# BITCOIN ANALYSIS
Bitcoin Analysis è un progetto per l’analisi avanzata dei dati legati a Bitcoin (BTC). Il repository è organizzato in moduli dedicati alla raccolta, elaborazione e visualizzazione dei dati.

| Cartella            | Descrizione                                                           |
| ------------------- | --------------------------------------------------------------------- |
| **DB**              | Contiene la struttura del database, tabelle, viste e i primi 10.000 blocchi in formato JSON |
| **PRE-PROCESSING**  | Contiene gli script per il caricamento e pulizia dei dati da JSON a database |
| **REPORTS**         | Contiene gli outputs, report (contenenti le varie anomalie) e grafici generati dopo l’elaborazione |
| **Unsupervised**    | Codice per tecniche *unsupervised learning* e cluster analysis |
| **WEBAPP**          | Frontend/back-end web basato su **Node.js** per esplorare i risultati |
