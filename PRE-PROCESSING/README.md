# PRE-PROCESSING
Operazioni effettuate prima di eseguire analisi sul database.

| Cartella            | Descrizione                                                           |
| ------------------- | --------------------------------------------------------------------- |
| **geolocation**     | Contiene gli script di gelocalizzazione degli indirizzi, un metodo basato sul fuso orario e un metodo al momento descritto solo in formato testuale |
| **insert_into_db**  | Contiene gli script di downloading dei blocchi tramite Bitcoin Core (via RPC) e inserimento a database locale postgresql  |
