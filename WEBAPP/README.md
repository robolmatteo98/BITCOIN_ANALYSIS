# WEBAPP
Un'applicazione WEB per visualizzare min formato tabellare e dettagliato, ogni anomalia trovata.
Attualmente le anomalie si trovano in formato JSON: backend/data/anomalies.json


| Cartella            | Descrizione                                                           |
| ------------------- | --------------------------------------------------------------------- |
| **backend**         | Contiene delle api richiamabili da frontend, elenco di anomalie e dettaglio di un'anomalia |
| **frontend**        | L'interfaccia dell'applicazione che mostra in formato tabellare le anomalie e i vari dettagli |
