# Modelli Unsupervised
Analisi della base di dati di Bitcoin.

| Cartella            | Descrizione                                                           |
| ------------------- | --------------------------------------------------------------------- |
| **classification**  | Contiene gli script di classificazione di un'anomalia in base ad alcuni parametri |
| **get_data**        | Contiene gli script di recupero dati da database, normalizzando alcuni parametri  |
| **utility**         | Contiene gli script di detection delle anomalie |
| **print_results**   | Contiene gli script di stampa degli outputs a video, dei report e grafici delle anomalie |
| **.**               | Contiene lo script main e le versioni precedenti del main |
